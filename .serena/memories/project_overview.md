# AyurSpace Flutter — Project Overview

**Purpose**: Flutter mobile app for Ayurvedic wellness. Users can discover plants, scan plants via camera, take dosha quizzes, chat with an AI assistant, and browse remedies.

**Tech Stack**:
- Flutter / Dart SDK ^3.5.0
- Firebase (Auth, Firestore) backend
- Riverpod state management (StateNotifierProvider)
- GoRouter navigation with auth guards
- Google Gemini AI + Plant.id API
- Google Fonts (Outfit)
- Localization: ARB files (English, Hindi, Marathi)

**Key entry point**: `lib/main.dart`
